Introducing Zed.js

Zed.js is a lightweight (Under 3kb) JQuery plug-in to assist with designing on the z-axis. This plug-in was developed as part of my final year project whilst I was studying BSc Digital Media at the University of the West of England. This site demonstrates the current capabilities of the plug-in through page navigation and mouse scroll. Zed.js works by applying 3d tranforms to sectioned content in a single page HTML document to add depth to the field of vision.

This plug-in is still in early development but hopefully it'll continue to grow and as issues are resolved, it may help contribute to a future design trend. Please feel free to contribute to it's future development through our GitHub repository, all pull requests will be reviewed.

Z-axis Design

Zed.js helps developer explore designing on the z-axis. The plug-in will enable content to be pushed behind each other and navigated in a 3D environment within the browser.

Web Technologies

Zed.js utilises modern web technologies and was developed in jQuery. Currently the plug-in only works on desktop platform but a mobile version is in the pipeline.

Browser Support

The plug-in works in most modern browsers and gracefully degrades the layout into a standard website where transitions are not supported and a single page site when JavaScript is disabled.

About Zed.js

This plug-in was developed as part of my final year project whilst I was studying BSc Digital Media at the University of the West of England. I was inspired to explore z-axis design for this project after reading Wren Lanier's article on A List Apart called, The Z-Axis: Designing for the Future. In this article Lanier explains that designing with system of layers will solve design problems when conserving space in the user interface. This led me to research and develop a solution to allow developers to explore the posibilities of designing for the z-axis in a 3D environment.

This site demonstrates the current capabilities of the plug-in through page navigation and mouse scroll. Zed.js works by applying 3d transforms to sectioned content in a single HTML page, to add depth to the field of vision. By adding in a sprinkle of the perspective property and a dash of transitions, the magic of Zed.js comes to life. Event listeners attached to the logo and each menu item initiate the functionality of the plug-in on mouse scroll or click.

The plug-in is a working progress and there are issues still to resolve. However, please feel free to contribute via our GitHub repository or send any comments, suggestions or a digital high five to email@zedjs.co.uk.

How to use Zed.js

To make Zed.js work in your project, start with a single page HTML document with sectioned content inside a wrapper. Then assign a class of zed to each section.

Next include the jQuery library and Zed.js plug-in just before the closing body tag and add zed.css underneath your existing stylesheet. Not forgetting to add the modernizr JavaScript library and HTML shiv for older browsers that don't support new CSS transitions or HTML5 tags. The stylesheet between the noscript tags cleans up the layout when JavaScript is disabled in the browser.

The the last step is to create your page navigation above your section wrapper. The scroll event listener (zedScroll) can be assigned to any element you like. We've assigned the listener to the whole of the header bar and it works perfect.

You may find you will need to adjust you existing CSS to suit the transitions. The plug-in can also work within existing page elements to develop an image gallery.

The following examples demonstrate the plug-in as an image gallery. Demo one contains a menu and scroll hotspot, in demo two the scroll listener is attached to each section, when the mouse is scrolled on the image the gallery changes. This approach works great on smaller page elements however it's not recommended for full pages as the user will not be ble to scroll down on the y-axis and content will be lost.
